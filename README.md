# Scida

## Beschreibung
Scida ist eine Webanwendung, die es Studenten und Dozenten erleichtern soll, Anwesenheit zu prüfen und nachzuhalten.
Scida digitalisiert den bisherigen Prozess der Laufzettel und erhöht die Sicherheit.
